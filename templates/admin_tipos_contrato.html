Pegue um contrato existente no Word, pinte de vermelho os dados que variam de contrato para contrato (nome do cliente, CPF, datas, valores, etc.). O sistema usa inteligência artificial para identificar o que cada texto vermelho representa e monta o template automaticamente.

Como funciona: se você pintar "Cassio Antônio da Silva" de vermelho, a IA reconhece pelo contexto que é o nome do contratante e substitui por {{ "{{" }} nome_contratante {{ "}}" }}. Tabelas de serviços com cabeçalho "Item / Descrição / Unidade / Valor / Prazo" são convertidas automaticamente para loop de serviços.
